  filter(!is.na(`GII Rank`)) %>%
  mutate(

Suggest a data plan

1. Source Identification
- Wikipedia: Gender Inequality Index (GII)
- Additional datasets: GDP, education (World Bank, UNDP, Gapminder, UNWOMEN, etc.)

2. Web Scraping Tools
- R packages: rvest for scraping, tidyverse for cleaning
- html_nodes() and html_table() to extract HTML tables

3. Cleaning Strategy
- Remove headers/footers
- Convert ranks and values to numeric
- Filter missing values

4. Analysis Plan
- Compare gender inequality by region
- Visualize GII vs GDP/education
- Highlight top/bottom ranked countries

5. Ethical Considerations
- Use public data only
- Document scraping code and source URLs
